This is how you make an unordered list:

  • Add an "ul" element (it stands for unordered list).
  • Add each item in its own "li" element (it stands for list element).
  • The order of list elements does not matter.

This is how you make an ordered list:

  1. Notice the new "ol" element wrapping the content (it stans for ordered list).
  2. But, the list item elements "li" are the same.
  3. Notice, also hot the number inside the list increment on their own.
  4. You should be noticing things in this precise order, because this is an ordered list. xD
